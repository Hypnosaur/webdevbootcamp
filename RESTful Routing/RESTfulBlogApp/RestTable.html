Restful Routes Table Page

The 7 RESTful Routes

Route Name URL Http Verb Purpose Mongoose Method
Index /dogs GET list all dogs Dog.find()
New /dogs/new GET Show new dogs form N/A
Create /dogs POST Create a new dog, then redirect somewhere Dog.create()
Show /dogs/:id GET Shows info about one specific dog Dog.findById()
Edit /dogs/:id/edit GET Show edit form for a dog Dog.findById()
Update /dogs/:id PUT Make changes to a dog and redirect Dog.findByIdAndUpdate()
Delete /dogs/:id DELETE Delete a dog and redirect Dog.findByIdAndRemove
